04 — Design Notes

Color Palette

Near-black #0a0a0a base · Electric lime #CAFF00 accent · Off-white #f5f5f0 text. One accent color, used sparingly = maximum impact. Never fill backgrounds with the accent — keep it as a signal.

Typography

Bebas Neue for display headings — loud, compressed, commanding. Space Mono for labels and metadata. Inter for body copy. This trio gives you editorial + technical + readable.

Animation Libraries

Framer Motion — your main tool. Page transitions, scroll animations, hover effects. Tailwind transitions for simple hovers. Add GSAP only if Framer Motion can't handle it. Keep the stack lean.

Culture Details

  • Crosshair cursor — gaming identity 🎮
  • Vinyl sleeve cards — music identity 🎵
  • Rotating hero subtitle — dev · designer · gamer
  • Interests section — hiphop, gaming, streetwear 👟
  • Footer easter egg — keep it subtle

Journal / MDX Setup

Use next-mdx-remote for blog posts. Store posts as .mdx files in /content/journal/. First post: "Building my portfolio from scratch — why I planned before I coded."

Deployment

Deploy to Vercel — built for Next.js, free tier, auto-deploys on every GitHub push. Connect a custom domain. Every push to main = live. Share that URL everywhere.

05 — Build Order
01

Init + global setup

Run npx create-next-app@latest — TypeScript + Tailwind + App Router. Install Framer Motion. Set up CSS variables. Push to GitHub immediately.

02

Navbar + crosshair cursor

Build app/layout.tsx with sticky navbar — initials logo left, links right. Add custom crosshair cursor as a Client Component. These live on every page, get them right first.

03

Home page — hero + stagger animation

Big name stagger, rotating subtitle with Framer Motion, two CTAs. Teaches you the difference between Server and Client Components — the animation requires "use client".

04

Projects page — vinyl cards + dynamic route

Build the vinyl sleeve card with CSS hover transition. Then tackle [slug] dynamic routing, generateStaticParams, and the case study layout. Core Next.js skills all in one page.

05

About page — scroll animations

Bio, animated skill bars using Framer Motion's useInView, interests strip. Teaches scroll-triggered animations and how to keep a page clean while still showing personality.

06

Journal — MDX setup

Install next-mdx-remote. Create /content/journal/. Build the list page and [slug] post page. Write your first post. No database needed — this is a great Next.js learning moment.

07

Contact + polish + deploy

Contact page with social tiles and form. Full design pass — spacing, consistency, details. Add page transitions. Deploy to Vercel. That live URL is your job application.
