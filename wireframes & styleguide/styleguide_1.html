06 — Voice & Tone

I build things that actually work — and look good doing it.
Confident, direct, no filler. Never "passionate developer seeking opportunities."

I'm [Name] — a developer based in [City]. I care about the details most people skip.

07 — Do / Don't
✓ Do
Use var(--accent) for all accent color references
One big animation moment per page — then micro-interactions only
Keep body copy in Inter — Bebas Neue is for headings only
Use lowercase for UI labels and nav links — feels more personal
Let the dark base breathe — generous padding and whitespace
Write like you talk — confident, casual, real
Tech stack tags use the accent border style — always
✗ Don't
Animate everything — it competes and overwhelms
Use pure black #000000 — use #0a0a0a for depth
Fill large areas with the accent color — it loses its punch
Use Inter for headings — that's what Bebas is for
Write "passionate" or "results-driven" anywhere on the site
Use more than 3 font weights — 400, 500, 700 only
Add drop shadows — the aesthetic is flat with border depth
